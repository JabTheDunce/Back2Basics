1.0 The Basics

HTML stands for HyperText Markup Language, this is a markup language which is designed to describe the layouts of elements in a web-page. Elements are described using tags and the syntax for a tag is as follows: <tag-name>. This is called an opening tag, and marks the start of an element, such as a paragraph, to close a tag it is the same, but it contains a leading forward-slash: </tag-name>, and all contents should go between these too for said element. Some examples of tags include: <html> wraps all HTML code, <head> wraps all meta-data stuff which isn't shown, <body> wraps all content which is shown, <p> starts a paragraph, <div> a container, etc.

1.1 Starting a document

To start a HTML document, you should start by writing <!DOCTYPE html>, this tells the browser that this is a HTML file, and lets it know how to parse it. You should follow this up with a set of html tags and then any markup for the web-page itself should go inside of here. Inside of this, you should add head and body tags, and use them as described above. This should give you a layout similar to this: <!DOCTYPE html>
<html>
 <head>
  <!-- Meta-data goes here! -->
 </head>
 <body>
  <!-- Page contents go here! -->
 </body>
</html>
 You can see I used <!-- comment --> to write text without showing it on the web-page. This can be used to annotate & comment on your own code, without affecting the code! You can now place anything you need to be displayed inside your body tags. Let's start by doing a basic header and some text. <!DOCTYPE html>
<html>
 <head>
  <!-- Meta-data goes here! -->
 </head>
 <body>
  <!-- Page contents go here! -->
  <h1>This is a large header!</h1>
  <p>This is a paragraph</p>
 </body>
</html>
 The <h1> tag is used for a large header, h stands for header and 1 is the largest size, instead of a 1, you could put any number from 1 to 6, to define a header of a different size, 6 being the smallest. <p> is used to define a paragraph, which is a much smaller font size.

2.0 Adding Stylesheets

CSS is used to style, position and format the elements described by your HTML, you can give an element a class or an ID using the following: <p class="paragraph" id="first-paragraph"></p>. This allows us to reference the element in CSS easily, in our new stylesheet named style.css, we can reference this paragraph in 3 different ways: p { /*this references all <p> elements in the code!*/
 color: red;
}

.paragraph { /*this references all classes named "paragraph"!*/
 text-align: center;
}

#first-paragraph { /*this references the single ID "first-paragraph"!*/
 font-size: 40px;
}
 Comments in CSS are also described using /* to open a comment and */ to close a comment. From the example, you can see the name of the element being addressed is written and then followed by a compound block denoted with braces {}, all styling and such goes inside the compound block, and the end of a line is terminated using ;. We only showcase some simple styling here, we first set all paragraph's (<p>) text colour to red, and follow by setting every .paragraph classed element to be aligned to the centre, and finally finish by setting the font size for all elements with an ID of #first-paragraph. This leaves our paragraph described above to have all of its content centered, large and red! But now we need to somehow link this to our HTML file for it to actually work, we can this using the <link> tag, this should be placed in your header as it is not content we want to be displayed for the user, it is meta-data and such which make our website work. <link rel="stylesheet" href="style.css"> You can see the rel tells the browser what this file actually is, and the href is used to share the location of the file, in this scenario we're placing our HTML file in the same directory as our style.css! If you return to your HTML file you should be able to see the result of this styling on any paragraphs! Hopefully, you can see how this can be used to style other elements in a similar fashion, these are not the only flags we have to control the style of an element, we could also set the height and width, add a background image or background colour, we could even set our own font, but to keep this tutorial relatively simple, I will leave these styles as an exercise to the reader!
